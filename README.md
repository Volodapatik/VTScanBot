# VTScanBot - VirusTotal Telegram Bot

Бот для сканирования файлов через VirusTotal.

## Функции
- Сканирование файлов любого размера
- Проверка URL и хешей
- Автоматическая загрузка больших файлов на Google Drive
- Очередь запросов (3 одновременно)

## Развертывание
1. Установите переменные окружения в Railway
2. Нажмите Deploy
